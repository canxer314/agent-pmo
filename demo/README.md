# Demo Gallery — Knowledge MEMO 作者的个人样本

> ⚠️ **警告：这些不是你应该复制的模板。**
>
> 以下 4 张卡片是作者个人领域（**投资 / 半导体周期 / 认知神经科学 / 分子生物学**）的**真实产物**，直接从作者的 Obsidian vault 里取出、做了最小化脱敏（去掉了一个内部研究报告 ID）。
>
> 它们展示的是"一个跑通的系统长什么样" —— 而不是"你的卡片应该长这样"。
>
> **知识管理是高度个人化的**。请看完这些样本之后，fork 这个 repo，删掉 `demo/`，从你自己真实写过的 4-7 张卡开始。

---

## 关于 wikilinks

你会注意到这些 demo 卡片里有很多 `[[wikilinks]]`（比如 `[[BDNF]]`、`[[HIIT]]`、`[[前额叶]]`、`[[多巴胺]]`）。这些链接指向作者 vault 里的其他卡 —— **它们没有被搬进这个 demo**，因为那会变成一整个 vault dump。

在你 clone 这个 repo 或在 Obsidian 里打开 demo/ 时，这些 wikilinks 会显示为 **紫色的空链**。这不是 bug —— 这是 [`../SCHEMA.md`](../SCHEMA.md) § 5 定义的 **知识缺口标记**：每一个紫色链接都是一个"我知道自己不知道"的点。

这本身就是 demo 的一部分：它告诉你**真实的知识网络是长什么样的** —— 永远有未填的空位，永远在扩展边界。

## Card 1: AI Hurtles Ahead — Howard Marks 的商业洞察

📁 [`cards/AI Hurtles Ahead — Howard Marks 2026-03-01.md`](./cards/AI%20Hurtles%20Ahead%20—%20Howard%20Marks%202026-03-01.md)

**类型**: Insight（洞察卡）
**领域**: AI 产业 / 价值投资
**来源**: Howard Marks 2026 年 2 月备忘录

**为什么它是一张好卡**：
- 示范了 **`/insights` 的三层提取结构**：每个洞察都有 "原文信号 → 深层逻辑 → 可迁移模式" 的推理链
- 7 个独立洞察用 `【】` 标记关键概念（`【自我改进闭环】`、`【量化Alpha之死】`、`【循环收入】` 等）—— 这些标记在 `/note` 的双提议阶段会被提议为独立的 atomic card
- 末尾的 "关键概念索引" 是提示 `/note` 的清单：这些都是候选原子卡
- 结尾的 "So What?" 部分展示了洞察卡和只是"摘要"的区别 —— 洞察卡必须有可执行的含义

**你可以从这张卡学到什么**：
- 如何把一篇 10 分钟的文章压缩成 7 个可迁移洞察
- 如何在保留推理链的同时避免"摘要型"平铺直叙
- `【】` 标记的实战用法（它们是作者早期的 atomic card 候选标记，v2 起会被 `/note` 识别）

## Card 2: MU 存储周期拐点研究

📁 [`cards/MU 存储周期拐点研究 2026-03-20.md`](./cards/MU%20存储周期拐点研究%202026-03-20.md)

**类型**: Research（研究摘要）
**领域**: 半导体产业周期 / 投资分析
**来源**: 多轮深度研究 + 历史数据交叉分析

**为什么它是一张好卡**：
- 示范了 **"研究摘要" 类 Card 的完整结构**：研究起点 → 历史周期复盘 → 当前定位 → 情景分析 → 验证指标 → 行动框架 → 研究局限
- **数据密度极高**：~230 行覆盖两个完整历史周期（2018, 2022）的 10+ 个关键指标
- 体现了 "human-in-the-loop" 的价值：给定完全相同的公开数据，一个 LLM agent 能生成这份研究吗？也许能生成 70%，但最后的**概率加权**（D=30%，P(已在顶部 ±1 个月)=15%）是**人基于直觉的主观判断**，这是 agent 无法替代的
- 末尾的 "研究局限" 部分展示了成熟研究员的态度：主动承认"仅 2 个完整周期，统计规律可能不稳健"

**你可以从这张卡学到什么**：
- 如何把一份财报 + 历史数据 + 当下观察组装成一份可行动的研究
- 如何设计 "主指标 + 辅助指标 + 观察日历"（研究的可验证性）
- 为什么 "研究局限" section 比看起来更重要：它强制你承认你不知道什么

## Card 3: 默认模式网络与意识

📁 [`cards/默认模式网络与意识.md`](./cards/默认模式网络与意识.md)

**类型**: Atomic（原子卡片，标签 `type/atomic`）
**领域**: 认知神经科学 / 意识研究
**来源**: 综合 Addis / Budson / Murray / Whyte 等多位学者论文

**为什么它是一张好卡**：
- 这是 **唯一一张 `type/atomic` 原子卡**，进了 `/review` 的 FSRS 间隔重复池
- **标签 `mastery/hard`** —— 这不是虚构的示范标签，而是作者**真实被这张卡反复难住**后 `/review` 打的标签
- 长度精确：24 行。足够完整呈现观点（意识 = 外显记忆/模拟的功能），足够短到一次复习可以全部消化
- 每段都用 `【】` 和 `[[wikilink]]` 标记关键概念
- 结尾的 `> 参考：...` 把 4 篇核心文献列出来 —— 原子卡可以精炼，但不能变成"没头没尾的断言"

**你可以从这张卡学到什么**：
- 原子卡的黄金长度和密度
- 如何在"精炼"和"可追溯"之间取得平衡（答案：参考文献）
- `type/atomic + mastery/hard` 组合的实际用法 —— 它告诉你作者在这张卡上栽过跟头

## Card 4: BDNF 深度研究

📁 [`cards/BDNF 深度研究 — 研究摘要.md`](./cards/BDNF%20深度研究%20—%20研究摘要.md)

**类型**: Research（研究摘要）
**领域**: 分子生物学 / 神经可塑性 / 个人健康
**来源**: 多轮深度调研（分子机制 → 脑功能 → 疾病 → 生活方式 → 饮食 → 脑力优化实操）

**为什么它是一张好卡**：
- 这是 **"跨层次整合型" 研究的样板** —— 从分子机制（mBDNF vs proBDNF 的"阴阳"系统）一路贯穿到"学习后 4 小时做 HIIT" 这类可执行建议
- 每个结论都有 **具体文献 + 样本量 + 效应量**（例：Erickson 2011 PNAS：120 人 RCT，1 年有氧 → 海马 +2%，对照组 -1.4%）
- 体现 "深度研究的 wikilink 密度" —— 大量 `[[BDNF]]`、`[[HIIT]]`、`[[多巴胺]]`、`[[地中海饮食]]`、`[[间歇断食]]` 互相交叉，形成一张本地知识网
- 结尾的 "关键结论" + "延伸问题" + "相关概念" 三段话是典型的研究摘要收尾：**总结这次学到什么 / 承认下次该学什么 / 连接到别的概念**

**你可以从这张卡学到什么**：
- 如何让"深度研究"既有分子级精度又有生活级可行动性
- wikilink 密度和知识图谱的正反馈（越多交叉，越容易调取）
- 从"阅读综述"到"可执行 protocol" 的转换路径

## 这 4 张卡为什么代表完整系统

这 4 张卡虽然只覆盖 3 个领域（投资 / 神经科学 / 分子生物学），但它们恰好展示了系统的**四种工作模式**：

| 卡片 | 工作模式 | 对应 Skill |
|---|---|---|
| AI Hurtles | 文章 → 洞察结构 | `/insights` |
| MU 周期 | 数据 + 历史 → 研究报告 | `/read` + `/note` |
| DMN | 综合文献 → 可复习原子卡 | `/note` + `/review` |
| BDNF | 多轮对话 → 跨层次整合 | `/read` + `/note` + 交叉 wikilink |

这不是巧合 —— 4 种工作模式是这套系统的四个主要 "use case"，而不是 4 种"卡片格式"。你在 fork 之后可能需要**不同的 4 种工作模式**，那 `demo/` 应该换成你自己的。
